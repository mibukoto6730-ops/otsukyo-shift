## 日曜・祝日ローテーション

### 薬剤師担当: B → C → D → B → …（自動）
- 固定休曜日と一致する祝日はその人をスキップ
  - B → 水曜祝日はスキップ
  - C → 火曜祝日はスキップ
  - D → 月曜祝日はスキップ
- 上書き: `sun_override_ph = {日付: "キー"}` で指定日だけ変更可

### 事務担当: H → I → J → H → …（自動）
- 固定休なし、単純ローテーション
- 上書き: `sun_override_jm = {日付: "キー"}` で指定日だけ変更可

### 日曜出勤の補償（週休2日維持）
| 担当 | 補償 |
|---|---|
| B / C / D / J | **前日土曜** をオフ |
| H / I | **翌月曜** をオフ（水曜シフト連続で6連勤防止） |

> 補償日は `*_extra_off` に自動追加される。祝日（平日）は補償なし。

### H の水曜担当週の月曜自動休み
- H（西山）は**第1・2水曜の週の月曜**を自動的に休みにする
- 月曜カバレッジ：H休 → I（早）＋ J（遅）で早番・遅番を確保
- I は水曜担当週（第3・4水）でも月曜は出勤（Mon→Tue→Wed→Thu→Fri = 最大5連勤）

## 金曜・土曜の配置ルール（B/C/D）

### 手動指定する場合
`b_fri`, `c_fri`, `d_fri`（金曜出勤日set）、`b_sat`, `c_sat`, `d_sat`（土曜出勤日set）に日付を記入。

### 自動配置（全て空欄の場合）
所定労働時間テーブルから「残り必要時間が多い2人」を各日に割り当てる。

- 土曜: 日曜補償でオフの日はスキップして、残り時間上位2人
- 金曜: 残り時間上位2人（最大5連勤なので問題なし）

## 毎月更新する変数

| 変数 | 内容 | 例 |
|---|---|---|
| `HOLIDAYS` | 臨時休業日の日付set | `{20}` → 20日が臨時休業 |
| `sun_override_ph` | 日祝薬剤師担当の上書き | `{5: "C"}` → 5日はC担当 |
| `sun_override_jm` | 日祝事務担当の上書き | `{12: "J"}` |
| `b_fri` / `c_fri` / `d_fri` | 金曜出勤日（空=自動） | `{4, 11, 18, 25}` |
| `b_sat` / `c_sat` / `d_sat` | 土曜出勤日（空=自動） | `{5, 19}` |
| `b_extra_off` 〜 `j_extra_off` | 追加休日（自動パターン外） | `{3}` |
| `requested_off` | 希望休（備考欄に「×」） | `{"B": {10}}` |
| `yukyu_per_person` | 有休（備考欄に「有休」） | `{"H": {22}}` |

## 希望休・有休（毎月更新）

```python
# 希望休（×表示）
requested_off = {k: set() for k in "ABCDEFGHIJK"}
# 例: requested_off["B"] = {10, 17}

# 有休（有休表示）
yukyu_per_person = {k: set() for k in "ABCDEFGHIJK"}
# 例: yukyu_per_person["H"] = {22}
```

## 検証チェック項目（スクリプト自動出力）

1. **日祝担当ローテーション** — 薬剤師・事務の割り当て一覧
2. **労働時間** — 合計h / 目標h / 差（±5h超で `!` フラグ）
3. **連勤チェック** — 最大連勤日数（6連勤以上で警告）
4. **薬剤師カバレッジ** — 曜日別要件チェック
5. **日祝配置** — 薬剤師1人・事務1人の確認
